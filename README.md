# SQL Queries

## 1st task Update το "Weight" Attribute 
###--> Please check the 1st_task_SQL_Query
  Περιγραφή: Προσθήκη "Weight" = "Thin" σε όλους τους εργαζομένους.
  Ελέγχει αν κάθε εργαζόμενος έχει ήδη ένα "Weight" attribute.
  Αν υπάρχει, ενημερώνεται το Weight value σε "Thin".
  Αν δεν υπάρχει, προστίθεται νέο attribute με "Weight" = "Thin".

## 2nd task Update το "Height" Attribute για τους Supervisors 
###--> Please check the 2nd_task_SQL_Query
